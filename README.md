# Digital Celebration Website

## Description

The Digital Celebration Website is a personalized web application built using **HTML, CSS, and JavaScript**. It is designed to make special occasions memorable by combining greetings, animations, music, images, and interactive effects in a single website. Users can enjoy an engaging and visually appealing celebration experience directly in their web browser.

## Features

* Personalized welcome page
* Birthday or celebration wishes
* Photo gallery and memories section
* Background music
* Animated effects and transitions
* Responsive and user-friendly design

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open **index.html** in any modern web browser.
